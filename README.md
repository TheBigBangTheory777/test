# Приложение для перемножения чисел

В этом проекте есть готовая страница с тремя числовыми полями ввода и кнопкой `Перемножить`. После нажатия на кнопку на странице показывается результат перемножения введённых чисел.

## Где находится проект

Откройте терминал и перейдите в папку проекта:

```bash
cd /workspace/test
```

## Как запустить страницу через командную строку

Выполните команды по очереди.

### 1. Установить зависимости

```bash
npm install
```

Эту команду нужно выполнить хотя бы один раз после получения проекта.

### 2. Запустить локальный сервер

```bash
npm start
```

После запуска сервер поднимает сайт на порту `3000`.

### 3. Открыть страницу в браузере

Откройте Chrome или другой браузер и введите в адресной строке:

```text
http://127.0.0.1:3000/index.html
```

### 4. Проверить работу кнопки

1. Введите числа в поля ввода.
2. Нажмите кнопку **«Перемножить»**.
3. Под кнопкой появится текст вида **`Результат: 24`**.

## Как запустить автоматический тест

Оставьте зависимости установленными и выполните:

```bash
npm run test:e2e
```

Тест автоматически:

1. открывает страницу;
2. нажимает кнопку;
3. проверяет, что появился правильный результат перемножения.

## Если страница не открывается

Проверьте по порядку:

1. Вы действительно выполнили `cd /workspace/test`.
2. Команда `npm install` завершилась без ошибки.
3. Команда `npm start` всё ещё запущена, и терминал с сервером не закрыт.
4. В браузере открыт именно адрес `http://127.0.0.1:3000/index.html`.

## Быстрая последовательность команд

Если нужна самая короткая инструкция, используйте только эти команды:

```bash
cd /workspace/test
npm install
npm start
```

После этого откройте:

```text
http://127.0.0.1:3000/index.html
```
